# Cybersecurity Awareness Chatbot

## Student Information
- Name: Taariq Flint
- Module: Programming (PROG6221)

## Project Description
This project is a C# console-based chatbot developed to educate South African citizens about cybersecurity threats such as phishing, weak passwords, and unsafe browsing practices.

## Features
- Voice greeting using a WAV file
- ASCII cybersecurity logo
- Interactive chatbot with personalised responses
- Input validation for incorrect or empty inputs
- Cybersecurity awareness topics:
  - Password safety
  - Phishing attacks
  - Safe browsing
- Console UI enhancements (colours, formatting)
- GitHub Actions Continuous Integration (CI)

## How to Run the Project
1. Open the project in Visual Studio
2. Build and run the application
3. Ensure the WAV file is in the correct folder
4. Follow the on-screen prompts

## CI Workflow
The project uses GitHub Actions for Continuous Integration.  
Every push triggers an automatic build to ensure the code runs correctly.
